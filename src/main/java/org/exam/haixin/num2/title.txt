【题目】TK 的暑假兼职利润计算  
【来源】海信集团 26 届秋招 Java 试卷  

=== 题目描述 ===
大学生 TK 趁暑假批发 n 件相同物品，每件批发价均为 x 元。  
随后 m 个人依次前来，第 i 个人出价 a_i 元购买一件。  
交易规则：若对方出价 ≥ y 元且仍有库存，则立即成交。  
求所有交易完成后 TK 的总利润：  
总利润 = 售出总金额 − 购买 n 件的总批发成本。

=== 输入格式 ===
多组测试数据。  
第一行一个整数 T（1 ≤ T ≤ 10^5），表示数据组数。  
对每组数据：  
  第一行 4 个整数 n m x y  
    n：库存件数 (1 ≤ n ≤ 10^9)  
    m：前来交易人数 (1 ≤ m ≤ 2×10^5)  
    x：每件批发价 (1 ≤ x ≤ 10^9)  
    y：最低接受价 (1 ≤ y ≤ 10^9)  
  第二行 m 个整数 a_1 … a_m（1 ≤ a_i ≤ 10^9），第 i 人出价。  
保证：单个文件所有组的 m 之和 ≤ 2×10^5。

=== 输出格式 ===
对每组数据，新起一行输出一个整数：总利润。

=== 样例输入 ===
3
3 4 10 15
20 12 18 25
2 1 5 8
9
12 10 50
60 40

=== 样例输出 ===
33
1
40

=== 提示 ===
按顺序遍历每人出价，若 ≥ y 且库存 > 0 则成交，库存减 1，累加售价。  
最后总利润 = 总售价 − n*x。  
复杂度 O(Σm)。
