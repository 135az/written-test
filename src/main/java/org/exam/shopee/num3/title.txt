如果字符串的所有字符出现的次数相同，则认为该字符串是有效的。
或者如果我们可以在字符串的某1个索引处删除1个字符，并且其余字符出现的次数相同，那么它也是有效的。
给定一个字符串inputStr，判断它是否有效。如果是，返回true，否则返回false。

备注：
1. 可以假设字符串仅包含小写字母
2. 0 <= length(inputStr) <= 10000

其他
时间限制: 3000ms
内存限制:256.0MB

输入输出示例
示例1
输入："shopee"
输出：true
说明：删除1个字母e，则满足要求

示例2
输入："lazada"
输出：false
说明：字母a出现3次，其它字母出现1次，不满足要求
